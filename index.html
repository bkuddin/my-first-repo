Git এবং GitHub সম্পর্কে আমি যা শিখলাম

সর্বপ্রথম আমি জানলাম যে Git হল একটি Source Code Version Control এবং Source Code Management System, অর্থাৎ আপনার কম্পিউটারে আগে কি কোড লিখেছেন এবং পরবর্তীতে তা কোথায় কোথায় পরিবর্তন করেছেন তার ভার্সন বা রিভিশন কন্ট্রোল এবং ভার্সন বা রিভিশন অনুযায়ী সংরক্ষণ করার একটি আদর্শ সিস্টেম ।


গিট নিয়ে কাজ শুরু করার জন্য আপনার কম্পিউটারকে গিট ব্যবহার উপযোগী করে নিতে হবে। সেজন্য আপনার কম্পিউটারে গিট ইন্সটল(install) করতে হবে। আপনি নিচে দেয়া git লোগোতে ক্লিক করুন এবং সেখান থেকে ডাউনলোড করে Install করুন।
